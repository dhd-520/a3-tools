# A3工具箱

一个用于管理A3账套的桌面工具，支持一键启动、数据库连接、远程访问等功能。

## 功能特性

### 1. 账套管理
- **账套列表**：显示所有A3账套信息
- **新增/编辑/删除**：完整的账套CRUD操作
- **搜索功能**：快速搜索账套代码、名称、地址
- **数据导入**：手动选择XML文件批量导入账套，自动生成编码，保留原始数据

### 2. 一键启动
- **电脑端**：启动君则A3.exe
- **开发工具**：启动君则A3集成开发工具.exe
- **网页版**：通过Chrome浏览器打开H5页面

### 3. 数据库连接
- 直接启动SSMS并连接到指定数据库
- 支持自动填充服务器、数据库名称、用户名
- 数据库密码自动解密并复制到剪贴板

### 4. 远程访问
- **RDP**：远程桌面连接，自动生成RDP文件
- **向日葵**：一键打开向日葵远程控制软件
- **其他**：复制远程地址到剪贴板

### 5. 进程管理
- 记录启动的所有进程ID
- 一键关闭所有已启动的进程

## 数据存储

### 账套数据 (accounts.json)
```json
{
  "code": "0001",
  "name": "账套名称",
  "server": "http://xxx.com/",
  "serverPassword": "账套密码(加密)",
  "database": "数据库地址",
  "databaseName": "数据库名称",
  "dbUser": "数据库用户名",
  "dbPassword": "数据库密码(加密)",
  "remoteType": "RDP/向日葵/其他",
  "remoteAddress": "远程地址",
  "remoteUser": "远程用户名",
  "remotePassword": "远程密码(加密)",
  "remark": "备注/原始XML数据"
}
```

### 设置数据 (settings.json)
```json
{
  "appDirectory": "A3应用程序目录",
  "launchDesktop": true,    // 启动电脑端
  "launchDevTools": true,     // 启动开发工具
  "launchWeb": false         // 启动网页版
}
```

## 安全机制

- **密码加密**：使用AES加密存储数据库密码、账套密码、远程密码
- **机器绑定**：加密密钥基于机器名和用户名生成
- **自动加解密**：保存时自动加密，加载时自动解密

## 使用说明

### 导入账套
1. 点击 **导入** 按钮
2. 在弹出的文件选择框中选择XML账套文件
3. 系统自动解析并导入数据
4. 自动生成新编码，密码加密存储
5. 原始XML数据保存到备注

### 启动账套
1. 在列表中选择账套
2. 点击 **启动** 按钮
3. 系统自动：
   - 修改AppConfig.xml中的服务器地址
   - 复制账套密码到剪贴板
   - 按设置启动相应程序

### 连接数据库
1. 选择账套
2. 点击 **链接数据库**
3. 自动启动SSMS并连接，密码复制到剪贴板

### 远程访问
1. 选择账套
2. 点击 **远程**
3. 根据远程方式自动执行RDP/向日葵/其他

## 开发说明

### 环境要求
- .NET 7.0 SDK
- Windows 10/11

### 编译运行
```bash
cd D:\work\A3Tools
dotnet build
dotnet run --project A3Tools\A3Tools.csproj
```

### 添加插件
1. 实现 `IPlugin` 接口
2. 编译为DLL
3. 放入 `Plugins\` 目录
4. 重启程序自动加载
